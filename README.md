# Only.You

Secure message book project with protected login and Vercel API support.

## Local setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run locally:
   ```bash
   node server.js
   ```
3. Open `http://localhost:3000`

## GitHub upload

The project is initialized as a git repository and committed locally.

To connect to GitHub, create a repository named `only.you` in your account, then run:

```bash
git remote add origin https://github.com/<YOUR_USERNAME>/only.you.git
git push -u origin main
```

## Vercel deployment

The project includes a Vercel configuration file (`vercel.json`) and an API route at `api/login.js`.

To deploy:
1. Push the project to GitHub.
2. Import the repository into Vercel.
3. Vercel will detect the Node API route and deploy automatically.

If you want, I can also help you create the repo URL or adjust the remote once you provide your GitHub username.
